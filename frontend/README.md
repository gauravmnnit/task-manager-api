# Task Manager Frontend

This is a minimal React + Vite + TypeScript frontend for the Task Manager API.

Features:
- React + TypeScript
- Vite dev server
- Tailwind CSS
- Axios API client
- React Query for server state

Setup

1. Install dependencies:

```powershell
cd frontend
npm install
```

2. Start dev server:

```powershell
npm run dev
```

The frontend expects the backend API at `http://localhost:3000` by default. You can change this in `frontend/.env` (VITE_API_URL).
